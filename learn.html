Printing a Message

Use print() function

print("Hello, world!")
  --> Hello, world!

print("Hello", "world!")
  --> Hello world!

print("Hello", 23)
  --> Hello 23

Comments

Use # for comments

# This is a comment

myString = "This side is not a comment"   # This side is a comment

myString = "# This is not a comment because it's inside quotes. So it is a string."

Variables

Variables do not need a declaration.

myVar = 5
print(myVar)
  --> 5

Variables can change type.

myVar = 10
myVar = True
myVar = "Hello"
print(myVar)
  --> "Hello"

Variable names can only contain letters(A-z), numbers(0-9), and underscores( _ )
and they cannot start with a number.

5myVar = 5
  --> Error

myVar$ = 8
  --> Error

my_Var5 = 10
print(my_Var5)
  --> 10

If statements

Whether if a line is in the if statement depends on indentation

if 5 > 2:
  print("I am inside the if statement")
  print("I am still inside the if statement")
print("I am not inside the if statement")

  --> I am inside the if statement
  --> I am still inside the if statement
  --> I am not inside the if statement

if 5 > 8:
  print("I am inside the if statement")
  print("I am still inside the if statement")
print("I am not inside the if statement")

  --> I am not inside the if statement

Use elif for else-if

score = 88
if score >= 90:
  print("A")
elif score >= 80:
  print("B")
elif score >= 70:
  print("C")
elif score >= 60:
  print("D")
else:
  print("F")

  --> B

Loops

while loops

i = 1
while i < 6:
  print(i)
  i += 1
print("Done")

  --> 1
  --> 2
  --> 3
  --> 4
  --> 5
  --> Done

for loops

food = ["hamburger", "pizza", "hotdog"]
for x in food:
  print(x)

  --> hamburger
  --> pizza
  --> hotdog

for x in range(5):
  print(x)

  --> 0
  --> 1
  --> 2
  --> 3
  --> 4

for x in range(2, 6):
  print(x)

  --> 2
  --> 3
  --> 4
  --> 5

Methods

def sum(x, y):
  result = x + y
  return result
print(sum(10, 8))
  --> 18

Classes

class Person:
  def __init__(self, name, age):
    self.name = name
    self.age = age

  def sayHello(self)
    print("Hello, I am", self.name)

myPerson = Person("Bob", 22)

print(myPerson.name)
print(myPerson.age)
myPerson.sayHello()

  --> Bob
  --> 22
  --> Hello, I am Bob
